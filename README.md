# Kinga
Kinga (Swahili for Protection) is a 'Safety by Design' PWA built for the 16 Days of Activism. Recognizing that Ethiopian girls facing digital violence often live in monitored homes, Kinga operates in 'Stealth Mode,' disguised as a calculator app.

Kinga ("Protection" in Swahili) is a **PWA (Progressive Web App)** that acts as a disguised crisis responder.
